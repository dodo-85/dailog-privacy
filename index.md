# DaiLog 개인정보 처리방침

**시행일: 2026년 5월 5일**

DaiLog(이하 "앱")는 이용자의 개인정보를 소중히 보호합니다. 본 개인정보 처리방침은 앱이 수집하는 개인정보의 항목, 수집 목적, 보유 기간 및 이용자의 권리에 대해 안내합니다.

---

## 1. 수집하는 개인정보 항목

앱은 서비스 제공을 위해 다음 정보를 수집합니다.

| 구분 | 수집 항목 | 수집 방법 |
|------|----------|----------|
| 필수 | 이메일, 이름, 프로필 사진 URL | Google 소셜 로그인 시 자동 수집 |
| 사용자 입력 | 활동 기록(블럭), 라벨, 목표, 설정값 | 앱 사용 중 이용자가 직접 입력 |
| 기기 정보 | FCM 토큰 (푸시 알림용), OS 버전, 기기 모델, 언어 설정 | 앱 설치 시 자동 생성 |
| 사용 분석 | 앱 사용 이벤트(화면 진입·주요 액션), 세션 정보(화면 터치·스크롤 녹화), 비정상 종료(크래시) 로그 | 앱 사용 중 자동 수집 (개인 식별 정보 제외, 익명 처리) |

앱은 민감정보(건강, 위치, 금융 등)를 수집하지 않습니다.

## 2. 개인정보 수집 및 이용 목적

| 목적 | 상세 | 동의 여부 |
|------|------|----------|
| 회원 식별 및 서비스 제공 | Google 로그인을 통한 사용자 인증, 데이터 저장 | 서비스 이용에 필수 (별도 동의 불필요) |
| 푸시 알림 발송 | 계획 완료 알림, 기록 리마인더, 빈 시간 알림 | 앱 내 설정에서 ON/OFF 가능 |
| 서비스 품질 개선 | 앱 사용 흐름 분석, 오류 발생 원인 파악, UX 개선을 위한 화면 사용 패턴 분석 | 익명 처리 (별도 동의 불필요) |

앱은 광고, 마케팅, 프로파일링 목적으로 개인정보를 사용하지 않습니다.

## 3. 개인정보 보유 및 파기

- **보유 기간**: 회원 탈퇴(계정 삭제) 시까지
- **파기 방법**: 회원 탈퇴 시 Firebase Authentication 계정 및 Firestore에 저장된 모든 사용자 데이터(블럭, 라벨, 목표, 설정)를 즉시 삭제합니다
- **자동 파기**: 별도의 보관 기간 경과 후 자동 삭제 정책은 없으며, 이용자 요청 시 삭제합니다

## 4. 개인정보의 제3자 제공

앱은 이용자의 개인정보를 제3자에게 제공하지 않습니다. 단, 다음의 경우는 예외입니다:

- 법령에 의한 요청이 있는 경우
- 이용자가 사전에 동의한 경우

## 5. 개인정보 처리 위탁 및 국외 이전

| 수탁자 | 위탁 업무 | 이전 국가 | 보호 조치 |
|--------|----------|----------|----------|
| Google LLC (Firebase Auth, Firestore, Cloud Messaging) | 사용자 인증, 데이터 저장, 푸시 알림 | 미국 | Google Cloud 표준계약조항(SCC) 적용 |
| Google LLC (Firebase Analytics, Crashlytics) | 앱 사용 분석, 크래시 로그 수집 | 미국 | Google Cloud 표준계약조항(SCC) 적용 |
| Microsoft Corporation (Clarity) | 세션 리플레이 및 히트맵 분석 | 미국 | Microsoft 표준계약조항(SCC) 적용 |

## 6. 이용자의 권리와 행사 방법

이용자는 다음 권리를 행사할 수 있습니다:

- **열람 요구**: 본인의 개인정보 처리 현황 확인
- **정정 요구**: 부정확한 정보의 수정
- **삭제 요구**: 개인정보 삭제 (계정 탈퇴)
- **처리 정지 요구**: 개인정보 처리 중단

## 8. 쿠키 및 행태 정보

앱은 쿠키를 사용하지 않습니다. 광고 식별자(ADID/IDFA)를 수집하지 않습니다.

세션 리플레이(Microsoft Clarity)는 화면의 터치·스크롤 등 조작 패턴을 익명으로 기록하며, 입력한 텍스트 등 민감 정보는 자동 마스킹 처리됩니다. 사용자 식별 정보는 함께 전송되지 않습니다.
